RELATIONAL CONSTRAINTS:
- Meeting -> Transcription (1:1)
- Meeting -> Summary (1:1)
- Meeting -> Conversation (1:N)
- Conversation -> Messages (1:N)
- Conversation -> Action Items (1:N)

CASCADE DELETE:
Delete Conversation
    -> Delete Messages
    -> Delete Action Items
    -> Delete Meeting
    -> Delete Transcription
    -> Delete Summary

================================================================================
                        ERROR HANDLING PATHS
================================================================================

File Too Large (>500MB)
    └-> HTTP 413 with size details

Compression Failed
    └-> HTTP 413 with error details

Transcription Error
    ├-> Rate Limit (429)
    └-> General (500)

Ollama Not Running
    ├-> Summary: returns error
    ├-> Action Items: returns error
    └-> But transcription still returned

Database Error
    ├-> Log [DB ERROR]
    ├-> Add to result["db_warning"]
    └-> Continue processing (non-fatal)

JSON Parse Error (Action Items)
    ├-> Return empty array
    ├-> Add warning field
    └-> Mark success=true (graceful)
